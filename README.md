A minimalist, high-performance eSIM catalog manager designed for travel affiliate websites. Create searchable catalogs of regional eSIM plans without the bloat.

== Description ==

The eSIM Affiliate Tool allows travel bloggers and SEO specialists to build beautiful, high-converting eSIM price comparison blocks (by Country) and a dropdown tool.

Designed with a minimalist aesthetic, it focuses on performance and user experience, avoiding the "plugin chaos" of heavier alternatives.

Features:
* Minimalist Editorial Design: Clean typography and high-contrast UI.
* Lightweight: Zero dependencies or heavy CSS frameworks.
* Shortcode Driven: Use [esim_search] for a search bar or [esim_catalog] for a full grid.
* Mobile-First: High-end modal previews for plan details.

== Installation ==

1. Upload the `esim-affiliate-tool` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'eSIM Tool' in your admin menu to add your first plans.
4. Use shortcodes to display the catalog on your post or page.
