A minimalist, high-performance eSIM catalog manager designed for travel affiliate websites. Create searchable catalogs of regional eSIM plans without the bloat.

== Description ==

The eSIM Affiliate Tool allows travel bloggers and SEO specialists to build beautiful, high-converting eSIM price comparison blocks and a searchable dropdown tool. 

Designed with a minimalist aesthetic, it focuses on performance and intentional user experience, avoiding the "plugin chaos" of heavier alternatives.

Features:
* **Minimalist Editorial Design:** Clean typography and high-contrast UI inspired by premium travel journals.
* **Lightweight:** Built for speed with zero heavy dependencies or frameworks.
* **Shortcode Driven:** Use `[esim_search]` for a destination search bar or `[esim_catalog]` for a full grid of available plans.
* **Mobile-First:** High-end modal previews optimized for travelers on the go.

== Installation ==

1. Upload the `esim-affiliate-tool` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'eSIM Tool' in your admin menu to add your first plans and affiliate links.
4. Use the provided shortcodes to display the tools on your posts or pages.
